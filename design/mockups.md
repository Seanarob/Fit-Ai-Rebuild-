# FitAI UI Mockups

## Design brief
- Dark-only theme with near-black background and lime-green accents
- Cards with soft glow/shadow, rounded corners, consistent padding
- Font hierarchy: prominent numbers/titles, concise copy, pill buttons
- Navigation: auth stack leading into bottom tab bar (Home, Workout, Nutrition, Progress)
- Components: `Card`, `StatRow`, `PrimaryButton`, `ProgressBar`, `SegmentedControl`, `ExerciseCard`, `PhotoGrid`
